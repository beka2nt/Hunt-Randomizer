## ğŸ‡ºğŸ‡¸ English Description

A simple web-based tool designed for *Hunt: Showdown 1896*. It generates random loadouts including weapons, tools, and consumables to add some chaos and fun (or suffering) to your bayou adventures.

> âš ï¸ **Current Version: v0.1**  
> This is an early beta version.

### âœ¨ Features
*   **Up-to-date Data**: Includes all purchasable and new weapons from the 1896 engine update (v2.6).
*   **Bilingual Support**: Toggle between English and Chinese (Simplified) instantly.
*   **Smart Logic**:
    *   Supports **Quartermaster** trait logic (Large Slot Primary + Medium Slot Secondary).
    *   Options to **Force Equip** Medkit and Melee tools (to keep it playable).
*   **Customizable**: Toggles to enable/disable randomization for Tools or Consumables.
